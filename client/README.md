# Carassauga Connect - Frontend

React + TypeScript PWA for the Portugal Pavilion Digital Experience.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- ✅ React 18 + TypeScript
- ✅ Vite for fast development
- ✅ Tailwind CSS for styling
- ✅ Lucide React for icons
- ✅ Axios for API calls
- ✅ Mobile-first responsive design

## Screens

1. **Home** - Live events, family status, quests
2. **Map** - Interactive pavilion map with family tracking
3. **Quests** - Quest progress and activities
4. **Profile** - Badges, passport, saved recipes

## Development

The frontend connects to the backend API at `http://localhost:4000/api`.

Make sure the backend server is running first!
